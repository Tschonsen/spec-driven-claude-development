# CLI project — SDCD template

Starter skeleton for a command-line tool project governed by SDCD.

## What this template gives you

- `CLAUDE.md` — project-level rules snippet (links to SDCD core rules)
- `design/UR_PLAN.md` — Ur-Plan template with required sections
- `design/CURRENT_STATE.md` — state file (baseline per §1.5)
- `design/.gitkeep`
- `.gitignore` — standard ignores for Python / Node / Rust

## Copy usage

When `/sdcd:new-project` scaffolds a CLI project, it copies this directory into the project root and fills in placeholders from the user's Ur-Plan answers.

## Placeholders

The template files use `{{project_name}}`, `{{stack}}`, `{{today}}` which the scaffolder substitutes.
